## Project Overview

This is a responsive Job Board UI built with React and Vite. It allows users to view job listings, search by title, filter by location/type, and includes bonus features like sorting and keyword highlighting.

## Features implemented

- **Job Cards**: Displays title, company, location, and job type.
- **Search**: Real-time filtering by job title with **Keyword Highlighting**.
- **Filters**: Filter by Location (Remote/On-site) and Type (Internship/Full-time).
- **Sorting**: Jobs are automatically sorted alphabetically.
- **Responsive Design**: Adapts seamlessly from mobile (1 column) to desktop (3 columns).

## How to Run Locally

1. Clone the repository.
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```nash
    npm run dev
    ```
